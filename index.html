The idea

Tarot × Pulse

The brand thesis is in the name. Tarot — ancient, gold, serif, the heirloom grimoire. Pulse — the signal, the data-line, the pattern-tracking that makes TarotPulse a category of one instead of one more pretty tarot journal. The look holds both at once: an heirloom grimoire that's secretly a data instrument. Every asset carries the same three tells — the gold pulse-waveform, the filigree that melts into circuit-traces, and constellations drawn as connected data-points.

Make it work

Building it in GoHighLevel

The mockups show the look; here's how to get that look into GHL without fighting it.

1 · Brand fonts & colors (do this once)

In the Funnel/Website builder, open Settings → Custom CSS (or the page's Custom Code/Head) and paste:

@import url('https://fonts.googleapis.com/css2?family=Cinzel:wght@600;700&family=Cormorant+Garamond:ital,wght@0,400;0,600;1,400&family=Space+Mono:wght@400;700&display=swap');

h1,h2,h3,.headline{font-family:'Cinzel',serif !important;color:#E2CF98;}
body,p,li{font-family:'Cormorant Garamond',serif !important;font-size:20px;}
.eyebrow,.label,button,.cta{font-family:'Space Mono',monospace !important;letter-spacing:1px;}

/* page */
body,.page{background:#16204a;color:#ECE7D8;}
a,button,.cta{background:#C8A44D;color:#1A1530;}

Brand color palette to set in the builder's color picker:

  • Background: #16204A (navy) / #0F1330 (deep sections)
  • Primary / buttons: #C8A44D (gold)
  • Headings on dark: #E2CF98 (soft gold)
  • Accent (sparingly): #B9A3E3 (lavender)
2 · Funnel structure
  1. Step 1 — Sales / Order page. Use a 2-step order form. Top half = the hero (headline + subhead + the cover image). Lower half = the offer stack + the gold "Add to Cart — Only $17" button + reassurance line.
  2. Step 2 — (optional) One-click upsell. If you ever add a deeper offer, this is where it goes. Skip for launch.
  3. Step 3 — Thank-you / Delivery page. Confirmation + download buttons (or trigger the delivery email). Same navy+gold skin.
3 · The product & delivery
  • Create a Product: "The Tarot Ritual Lab" — price $17 one-time. Product image = cover.jpg.
  • Bundle the three deliverables (Journal PDF + Audio + Symbol Map) as the fulfillment — attach files to the product, or deliver via the thank-you page download buttons, or an automation email.
  • Connect your Stripe/payment integration to the order form.
  • Tip: a small "order bump" works well here if you split the Audio Guide out as the bump (+$X) — but the clean $17-all-in is the stronger launch.
4 · Assets to upload & mobile
  • Upload cover.jpg (product + hero), audio-guide.jpg and symbol-map.png (offer-stack visuals / deliverables).
  • Use the sales-hero image only as a layout reference — rebuild it with real text blocks so it stays editable and crisp.
  • Mobile: stack the sections, keep the gold-on-navy contrast, and make the CTA button full-width. Keep one pulse-divider between sections — don't over-decorate small screens.
